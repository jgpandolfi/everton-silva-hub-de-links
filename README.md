## 💡 Sobre o Projeto

Este é um projeto real desenvolvido para o cantor **Everton Silva**, vencedor da última temporada do programa Ídolos em 2012. O website funciona como um **hub centralizado** que concentra todos os links e formas de contato do artista, implementando conceitos modernos de desenvolvimento web full-stack.

### 🎯 Objetivos Principais

- **Hub de Links Profissional**: Centralizar todas as redes sociais e contatos do artista
- **Experiência Premium**: Design elegante que reflete a identidade visual do artista
- **Analytics Avançado**: Sistema completo de tracking e análise de visitantes
- **Gestão Administrativa**: Dashboard para equipe de marketing e administração
- **Conformidade Legal**: Total adequação à LGPD e políticas de privacidade

---

## 🚀 Funcionalidades

### 🌟 Frontend (Website Principal)

- **🎨 Design Imersivo**: Interface elegante com animações sutis e cursor personalizado
- **📱 Totalmente Responsivo**: Adaptação perfeita para todos os dispositivos
- **🔗 Hub de Links Sociais**: Instagram, YouTube, TikTok e WhatsApp com contagem de cliques
- **📅 Carrossel de Eventos**: Sistema dinâmico que exibe apenas eventos futuros
- **📋 Formulário de Contato**: Validação em tempo real com integração WhatsApp
- **🍪 Modal LGPD**: Sistema completo de consentimento e políticas de privacidade
- **⚡ Preloader Interativo**: Carregamento otimizado com barra de progresso
- **🎯 Tooltips Informativos**: Dicas contextuais para melhor usabilidade

### 🛠️ Backend (API e Processamento)

- **📊 Sistema de Tracking**: Rastreamento completo de visitantes com métricas avançadas
- **📝 Processamento de Formulários**: Validação, sanitização e armazenamento seguro
- **🔔 Notificações em Tempo Real**: Webhooks Discord para alertas instantâneos
- **🌍 Geolocalização por IP**: Identificação automática de localização dos visitantes
- **📈 Métricas de Engagement**: Duração de sessão, cliques e comportamento do usuário
- **🔐 Sistema de Autenticação**: Login seguro com hash Argon2ID e rate limiting

### 📊 Dashboard Administrativo

- **📈 Visualização de Dados**: Gráficos interativos com Chart.js
- **📋 Gestão de Leads**: Visualização e filtros avançados de contatos
- **🌎 Analytics Geográfico**: Mapa de visitantes por país e região
- **📅 Filtros Temporais**: Análise de dados por períodos customizados
- **👥 Gestão de Usuários**: Sistema de permissões para admin e marketing
- **📊 Relatórios Detalhados**: Estatísticas completas de performance do site

## 📊 Dashboard Administrativo

## 🔒 Segurança e Conformidade

### 🛡️ Medidas de Segurança Implementadas

- **Sanitização de Dados**: Validação e limpeza de todas as entradas
- **Proteção CSRF**: Tokens únicos para prevenção de ataques
- **Headers de Segurança**: CSP, HSTS e outras proteções
- **Validação de Origem**: Verificação de referer para APIs sensíveis
- **SQL Injection Protection**: Prepared statements em todas as queries

### 📋 Conformidade LGPD

- **Consentimento Explícito**: Modal interativo na primeira visita
- **Política de Privacidade**: Documento completo e acessível
- **Gestão de Cookies**: Carregamento condicional de tracking
- **Direitos do Usuário**: Transparência sobre coleta e uso de dados
- **Armazenamento Seguro**: Dados sensíveis protegidos e criptografados

## 📈 Analytics e Tracking

### 📊 Métricas Coletadas

**Dados do Visitante:**
- IP e geolocalização (país, estado, cidade)
- Dispositivo (marca, sistema operacional, navegador)
- Tempo de permanência no site
- Número total de cliques realizados
- Identificação de novo visitante vs retornante

**Dados de Engagement:**
- Cliques em cada rede social
- Preenchimento de formulários
- Parâmetros UTM de campanhas
- Origem do tráfego (referrer)

**Dados Técnicos:**
- Resolução de tela
- User agent completo
- Provedor de internet
- Timestamp de todas as ações

### 🔔 Notificações em Tempo Real

- **Novos Visitantes**: Webhook Discord com dados resumidos
- **Leads Gerados**: Notificação completa via Discord
- **Picos de Tráfego**: Alertas automáticos para volumes altos
- **Erros do Sistema**: Logs automáticos para monitoramento
